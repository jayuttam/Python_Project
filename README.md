# 📊 Retail Sales Forecasting Dashboard

An **end-to-end Retail Sales Forecasting application** built using **Python, Streamlit, and Machine Learning** to analyze historical retail data, generate actionable sales insights, and forecast future sales trends.

This project combines **data analysis, interactive visualization, and predictive modeling** to support data-driven business decision-making.

---

## 🚀 Project Features

- 📈 Interactive sales insights (daily trends, product categories, customer segments)
- 🧹 Data preprocessing and feature engineering from raw retail data
- 🤖 Machine learning–based sales prediction using **Random Forest** and **XGBoost**
- 📊 Model evaluation using **MAE** and **RMSE**
- 🔮 Future sales forecasting with interactive visualizations
- 🖥️ User-friendly dashboard built with **Streamlit**

---

## 🛠️ Tech Stack

- **Python**
- **Pandas, NumPy** – Data manipulation
- **Scikit-learn** – Model training & evaluation
- **XGBoost** – Advanced regression modeling
- **Streamlit** – Interactive web dashboard
- **Plotly** – Dynamic visualizations
